# version - Number

  • Props:
    • embeddedFile- String | EmbeddedFile | List<String> | List<EmbeddedFile>
    • name- String, defaults to "null"
  • Returns: Document

# addPages(Number pageNumber, Page | List<Page> page=null)

Add page at specified location

  • Props:
    • pageNumber- Number
    • page- Page | List<Page>, defaults to "null"
  • Returns: List<Page>

# addWatermarkToPages(Page overlayPage, Position position=BACKGROUND, Number | List<Number> pageNumbers=null)

Add overlay page as watermark to full document or list of pages

  • Props:
    • overlayPage- Page
      any page can be used as overlay
    • position- Position, defaults to "BACKGROUND"
      FOREGROUND or BACKGROUND
    • pageNumbers- Number | List<Number>, defaults to "null"
      null means whole document
  • Returns: Document

# resizePages(PaperSize | Number size, Boolean resizePage=true, Boolean resizeContent=true, BasePoint basePoint=CENTER, Number | List<Number> pageNumbers=null)

Resize document or list of pages

  • Props:
    • size- PaperSize | Number
    • resizePage- Boolean, defaults to "true"
    • resizeContent- Boolean, defaults to "true"
    • basePoint- BasePoint, defaults to "CENTER"
    • pageNumbers- Number | List<Number>, defaults to "null"
      null every page in document
  • Returns: Document
