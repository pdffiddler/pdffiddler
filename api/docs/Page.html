# addImage(String | Image image, Point | Region at=null, Number width=null, Number height=null, Boolean inline=false, GState gs=null)

Add image at particular location

  • Props:
    • image- String | Image
      Can be String/Image type
    • at- Point | Region, defaults to "null"
      Can be Point/Region type
    • width- Number, defaults to "null"
      default null, either derive from Region or Actual Width
    • height- Number, defaults to "null"
      default null, either derive from Region or Actual Height
    • inline- Boolean, defaults to "false"
      if true, then image will drawn as inline
    • gs- GState, defaults to "null"
  • Returns: Page
