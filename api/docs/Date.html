# Date

# Methods

# format(String pattern)

Format date using below pattern

  • Props:
    • pattern- String
  • Returns: String

# minus(Number amountToSubtract, ChronoUnit unit)

Minus field from date

  • Props:
    • amountToSubtract- Number
    • unit- ChronoUnit
  • Returns: Date

# now()

Return current date

  • Returns: Date

# parse(String dateStr, String pattern)

Parse String using pattern

  • Props:
    • dateStr- String
    • pattern- String
  • Returns: Date

# plus(Number amountToAdd, ChronoUnit unit)

Plus field to date

  • Props:
    • amountToAdd- Number
    • unit- ChronoUnit
      eg: HOURS, MINUTES, SECONDS, DAYS, WEEKS, MONTHS, YEARS
  • Returns: Date
